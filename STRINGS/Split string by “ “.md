# Split string by “ “ : Combine with the C++ Folder string manipulation file\n\n```cpp\n#include <sstream>\n#include <vector>\n#include <string>\n\nusing namespace std;\n\nvector<string> splitByWhitespace(const string& str) {\n    vector<string> tokens;\n    istringstream iss(str);\n    string token;\n    while (iss >> token) {\n        // Reads words separated by whitespace\n        tokens.push_back(token);\n    }\n    return tokens;\n}\n```
